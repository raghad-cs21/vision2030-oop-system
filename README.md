# Vision 2030 Smart System (OOP Project)

This project is a Java-based object-oriented system designed to demonstrate core OOP concepts such as inheritance, polymorphism, and composition.

## Features
- Person base class
- Instructor class extending Person
- Course management system
- Main driver class for system execution

## OOP Concepts Demonstrated
- Inheritance
- Polymorphism
- Encapsulation
- Composition

## Technologies Used
- Java
- Object-Oriented Programming

## Purpose
This project was developed as part of an Object-Oriented Programming course to demonstrate how OOP principles can be used to design structured software systems.
